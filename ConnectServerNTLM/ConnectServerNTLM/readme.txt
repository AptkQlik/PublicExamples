Qlik Sense .Net SDK 4.0
=========================

The Qlik Sense .NET SDK is a development toolbox for use with applications such as Microsoft� Visual Studio�. 

The Qlik Sense .NET SDK can be used for building a Qlik Sense app from scratch or for visualizing an 
existing app in a custom client by embedding Qlik Sense objects. It can also be used for administering 
or modifying Qlik Sense apps in many different ways or for using Qlik Sense as a computing device.

Applications, in which the Qlik Sense .NET SDK library is embedded, can be written in any .net language.
